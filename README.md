# NumPy Data Analysis Toolkit

A simple Python toolkit built using NumPy for performing data analysis on CSV datasets.

## Features
- Load CSV data
- Handle missing values
- Calculate mean, median, std deviation
- Normalize data
- Generate correlation matrix

## Installation
pip install -r requirements.txt

## Usage
python src/main.py

## Example
Enter CSV file path: data/sample_data.csv

## Tech Stack
- Python
- NumPy
